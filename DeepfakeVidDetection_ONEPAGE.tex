\section{Method Overview}
We leverage \textbf{DINOv3 ViT-B/16} with strategic LayerNorm fine-tuning for deepfake detection. Key innovations: (1) \textit{LayerNorm-specific tuning}---freeze all parameters except LayerNorm modules and final transformer block, preserving pre-trained representations while adapting to deepfake artifacts; (2) \textit{Metric learning} with angular margin ($m=0.6$) pushing real/fake embeddings apart; (3) \textit{Compression-aware training} with JPEG augmentation (quality 50--90) to match real-world video distributions.

\textbf{Architecture:} 768-dim DINOv3 embeddings $\rightarrow$ L2-normalized linear probe (1,538 params). MTCNN face detection with vertical shift compensation optimizes crop regions for artifact detection.

\section{Key Findings}
\begin{itemize}[leftmargin=*,itemsep=1pt,topsep=2pt]
    \item \textbf{Transfer learning advantage:} LayerNorm tuning achieves superior accuracy with only 0.5M trainable parameters vs. 20M+ for full fine-tuning
    \item \textbf{Speed-accuracy trade-off:} DINOv3 provides optimal balance---faster than Xception (8ms vs 12ms) with significantly higher accuracy (0.88 vs 0.81 AUROC)
    \item \textbf{Distribution matching:} JPEG augmentation prevents -0.05 AUROC degradation on compressed videos
    \item \textbf{Metric learning impact:} Angular margin loss improves embedding discrimination for ambiguous cases
\end{itemize}

\section{Implementation Notes}
\textbf{Training:} FaceForensics++ (c23 quality), paired sampling, batch size 32, AdamW optimizer. \textbf{Augmentation:} JPEG compression (50--90, $p$=0.3), Gaussian blur ($\sigma$=0.1--2.0, $p$=0.2), color jitter, rotation ($\pm15°$). \textbf{Inference:} JPEG simulation (quality=90) applied to match training distribution. Code available: \texttt{github.com/aryanbiswas16/DeepfakeVidDetection}
